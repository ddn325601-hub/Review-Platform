# 代码导读

## 后端分层

| 层 | 路径 | 说明 |
| --- | --- | --- |
| Controller | `src/main/java/com/hmdp/controller` | 接收前端请求，处理参数和响应 |
| Service 接口 | `src/main/java/com/hmdp/service` | 业务能力定义 |
| Service 实现 | `src/main/java/com/hmdp/service/impl` | 核心业务逻辑 |
| Entity | `src/main/java/com/hmdp/entity` | MySQL 表映射对象 |
| DTO | `src/main/java/com/hmdp/dto` | 前后端传输对象 |
| Mapper | `src/main/java/com/hmdp/mapper` | MyBatis-Plus 数据访问 |
| Utils | `src/main/java/com/hmdp/utils` | Redis key 常量、缓存工具、锁、拦截器、ID 生成器 |

## Redis 主题索引

| 主题 | 主要文件 | 观察点 |
| --- | --- | --- |
| 验证码 TTL | `UserServiceImpl` | `LOGIN_CODE_KEY + phone`，验证码过期时间 |
| 登录态共享 | `UserServiceImpl`, `RefreshTokenInterceptor` | token hash 存用户，拦截器刷新 TTL |
| 缓存穿透 | `CacheClient#queryWithPassThrough` | 空字符串缓存空结果 |
| 缓存击穿 | `CacheClient#queryWithMutex` | 互斥锁重建缓存 |
| 逻辑过期 | `CacheClient#queryWithLogicalExpire` | 返回旧数据，后台线程重建 |
| Redis 分布式 ID | `RedisIdWorker` | 时间戳 + Redis 自增序列 |
| Lua 秒杀 | `seckill.lua` | 扣库存、一人一单、写 Stream 原子执行 |
| Stream 异步订单 | `VoucherOrderServiceImpl` | 消费者组、ack、pending list |
| Redisson 锁 | `RedissonConfig`, `VoucherOrderServiceImpl` | 一人一单落库前按用户维度加锁 |
| GEO 附近商户 | `ShopServiceImpl#queryShopByType` | GEOSEARCH + 距离回填 |
| BitMap 签到 | `UserServiceImpl#signCount` | 右移统计连续签到 |
| ZSet 点赞排行 | `BlogServiceImpl#queryBlogLikes` | top5 点赞用户按时间排序 |
| Feed 流 | `BlogServiceImpl#queryBlogOfFollow` | ZSet 滚动分页 |
| Set 共同关注 | `FollowServiceImpl#followCommons` | Redis set 交集 |

## 起始版和完整版对照

查看两个后端目录差异：

```powershell
git diff --no-index backend\hm-dianping-starter backend\hm-dianping-complete
```

按单个业务文件查看差异：

```powershell
git diff --no-index backend\hm-dianping-starter\src\main\java\com\hmdp\service\impl\UserServiceImpl.java backend\hm-dianping-complete\src\main\java\com\hmdp\service\impl\UserServiceImpl.java
```

## 兼容性调整

- 两个后端的 Lombok 版本已升级到 `1.18.32`，用于兼容较新的 JDK。
- 完整实现版 `VoucherOrderServiceImpl` 中 Redis Stream 的 ACK 目标为 `stream.orders`。
- 图片上传目录通过 `hmdp.upload.dir` JVM 参数配置，默认使用 `uploads/`。
