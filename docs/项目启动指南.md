# 项目启动指南

## 当前推荐启动方式

当前工作区已经按你的机器调整为：

| 服务 | 地址 | 账号 | 密码 |
| --- | --- | --- | --- |
| MySQL | `127.0.0.1:3307` | `root` | `l798267901` |
| Redis | `127.0.0.1:6380` | 无 | `l798267901` |
| 后端起始版 | `http://127.0.0.1:8081` | - | - |
| 前端 Nginx | `http://127.0.0.1:8082` | - | - |

浏览器访问：

```text
http://127.0.0.1:8082
```

## 一键启动起始版

在仓库根目录执行：

```powershell
.\scripts\start-starter.ps1
```

停止本项目启动的服务：

```powershell
.\scripts\stop-local.ps1
```

## 修改为你自己的账号密码

默认配置在两个文件中：

```text
backend/hm-dianping-starter/src/main/resources/application-local.yaml
backend/hm-dianping-complete/src/main/resources/application-local.yaml
```

它们支持环境变量覆盖。复制示例文件：

```powershell
Copy-Item .\config\local-env.example.ps1 .\config\local-env.ps1
```

然后编辑 `config/local-env.ps1`：

```powershell
$env:HMDP_MYSQL_HOST = "127.0.0.1"
$env:HMDP_MYSQL_PORT = "3307"
$env:HMDP_MYSQL_DATABASE = "hmdp"
$env:HMDP_MYSQL_USERNAME = "root"
$env:HMDP_MYSQL_PASSWORD = "l798267901"

$env:HMDP_REDIS_HOST = "127.0.0.1"
$env:HMDP_REDIS_PORT = "6380"
$env:HMDP_REDIS_PASSWORD = "l798267901"
```

如果你要连接原本的 MySQL `3306`，把端口、用户名和密码改成你的即可。

## 手动启动后端

起始版：

```powershell
cd backend\hm-dianping-starter
mvn spring-boot:run -Dspring-boot.run.profiles=local
```

完整实现版：

```powershell
cd backend\hm-dianping-complete
mvn spring-boot:run -Dspring-boot.run.profiles=local
```

注意：完整实现版的秒杀异步队列依赖 Redis Stream。你当前本机 Redis 是 `3.2.100`，不支持 Stream；学习完整秒杀章节时建议换 Redis 5+。

## 验证

后端直连：

```powershell
Invoke-WebRequest -UseBasicParsing http://127.0.0.1:8081/shop-type/list
```

前端代理：

```powershell
Invoke-WebRequest -UseBasicParsing http://127.0.0.1:8082/api/shop-type/list
```

看到 `success=true` 即说明链路正常。
