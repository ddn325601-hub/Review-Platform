# 本地复现指南

本文提供从依赖准备到前后端联通的完整复现流程。

## 1. 准备环境

建议安装：

- JDK 8 或 11
- Maven 3.6+
- MySQL 5.7+
- Redis 5+
- Nginx 或其他静态服务工具

也可以使用仓库中的 `docker-compose.yml` 启动 MySQL 与 Redis。

## 2. 启动 MySQL 和 Redis

使用 Docker Compose：

```powershell
docker compose up -d
docker compose ps
```

首次启动时，MySQL 会导入：

```text
resources/db/hmdp.sql
```

Docker Compose 中的默认服务配置以 `docker-compose.yml` 为准。如需连接已有 MySQL/Redis，请使用环境变量覆盖。

## 3. 配置本地连接

复制本地配置模板：

```powershell
Copy-Item .\config\local-env.example.ps1 .\config\local-env.ps1
```

编辑后加载：

```powershell
. .\config\local-env.ps1
```

`application-local.yaml` 会读取以下环境变量：

```text
HMDP_MYSQL_HOST
HMDP_MYSQL_PORT
HMDP_MYSQL_DATABASE
HMDP_MYSQL_USERNAME
HMDP_MYSQL_PASSWORD
HMDP_REDIS_HOST
HMDP_REDIS_PORT
HMDP_REDIS_PASSWORD
```

## 4. 初始化 Redis Stream

完整实现版的异步秒杀订单使用 Redis Stream。首次运行前执行：

```powershell
docker exec -it hmdp-redis redis-cli -a 123321 XGROUP CREATE stream.orders g1 0 MKSTREAM
```

如果返回 `BUSYGROUP`，表示消费组已经存在。

## 5. 运行后端

完整实现版：

```powershell
cd backend\hm-dianping-complete
mvn spring-boot:run -Dspring-boot.run.profiles=local
```

起始版：

```powershell
cd backend\hm-dianping-starter
mvn spring-boot:run -Dspring-boot.run.profiles=local
```

后端默认监听：

```text
http://127.0.0.1:8081
```

## 6. 运行前端

前端目录：

```text
frontend/hmdp
```

前端页面中的接口请求默认走 `/api`。推荐通过 Nginx 托管静态页面，并配置 `/api` 代理到后端。

示例：

```nginx
server {
    listen 8082;

    location / {
        root /path/to/frontend/hmdp;
        index index.html;
    }

    location /api {
        rewrite /api/(.*) /$1 break;
        proxy_pass http://127.0.0.1:8081;
    }
}
```

浏览器访问：

```text
http://127.0.0.1:8082
```

直接打开 `frontend/hmdp/index.html` 可以查看静态页面，但不会自动获得 `/api` 反向代理。

## 7. 验证接口

商户分类：

```powershell
curl "http://127.0.0.1:8081/shop-type/list"
```

发送验证码：

```powershell
curl "http://127.0.0.1:8081/user/code?phone=13800138000"
```

导入附近商户 GEO 数据：

```powershell
cd backend\hm-dianping-complete
mvn -Dtest=HmDianPingApplicationTests#loadShopData test -Dspring.profiles.active=local
```

秒杀压测脚本：

```text
resources/jmeter/秒杀抢购.jmx
```

## 8. 常见问题

`RedisConnectionFailureException`：

- 确认 Redis 正在运行。
- 确认后端启动时使用了 `local` profile。
- 确认 `HMDP_REDIS_PASSWORD` 与 Redis 实际密码一致。

`Access denied for user 'root'`：

- 确认 MySQL 用户名、密码、端口和数据库名正确。
- 如使用 Docker Compose，确认容器与 volume 属于当前项目。

秒杀接口返回成功但订单未落库：

- 确认 Redis 版本支持 Stream。
- 确认已创建 `stream.orders` 的 `g1` 消费组。
- 查看后端日志中的 Stream 消费异常。
